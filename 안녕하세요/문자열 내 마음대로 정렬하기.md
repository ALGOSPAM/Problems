```c#
using System;

public class Solution {
    public string[] solution(string[] strings, int n) {
        string[] answer = new string[strings.Length];
        Array.Copy(strings, answer, strings.Length);
        Array.Sort(answer, (string a, string b) => {
                if (a[n] == b[n])
                {
                    return a.CompareTo(b);
                }
                return a[n] < b[n] ? -1 : 1; 
        });
        return answer;
    }
}
```

시간복잡도

- **O(nlogn)**
